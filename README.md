# DistuBot
A cursed Discord bot, not for actual use. It's cursed, so ignore the odd facts or images.

----------------------------------------

A few notes:
 - Made using replit.com, an online python IDE but way easier to use.
 - keep_alive.py was made by a tutorial, aswell as the very basics of main.py.
 - The .env file isn't included (hidden value for my bot token, it's the password for the bot)

 - The images are links due to them being able to be embedded with Discord, resulting in a way easier life.
 - Due to complications, the bot only stays up for an hour unless interacted with. I know how to fix it, although it is unneeded currently.

----------------------------------------

Known issues:
 - When using ~d repeat you can no longer ask DistuBot to say "repeat", nor any characters used in the command
 - DistuBot frequently repeats it's facts and images when using ~d cursed fact / ~d cursed image (due to the lack of content)
